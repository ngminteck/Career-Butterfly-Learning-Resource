The OpenGL Extension Wrangler Library (GLEW) is a cross-platform open-source C/C++ extension loading library. It provides efficient run-time mechanisms for determining which OpenGL extensions are supported on the target platform. GLEW exposes both core and extension functionality in a single header file. This library has been tested on various operating systems, including Windows, Linux, Mac OS X, FreeBSD, Irix, and Solaris123.

In summary:

  • Purpose: To manage OpenGL extensions and provide a consistent interface for accessing them.
  • Features:
    • Cross-platform: Works on multiple operating systems.
    • Open-source: Available for community use and contributions.
    • Efficient: Provides runtime mechanisms to determine supported extensions.
    • Single header file: Exposes both core and extension functionality.
  • Supported OpenGL Versions: The latest release (version 2.1.0) includes support for OpenGL 4.6, compatibility, and forward-compatible contexts1.
