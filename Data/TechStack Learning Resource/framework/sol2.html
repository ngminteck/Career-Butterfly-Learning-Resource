sol2 is a C++ library binding to Lua, designed to facilitate seamless interaction between C++ code and Lua scripts. Here are some key points about sol2:

  1. Purpose and Features:

    • Binding: It bridges the gap between C++ and Lua, allowing you to call Lua functions from C++ and vice versa.
    • Ease of Use: sol2 aims to be user-friendly and straightforward to integrate into projects.
    • Header-Only: The library is header-only, making it easy to include in your projects without additional dependencies.
    • Lua Versions: It supports all Lua versions 5.1+, including LuaJIT 2.0+ and MoonJIT.

  2. Usage Examples:

    • Basic Function Binding:
      #include <sol/sol.hpp>
#include <cassert>

int main() {
    sol::state lua;
    int x = 0;
    lua.set_function("beep", [&x] { ++x; });
    lua.script("beep()");
    assert(x == 1);
}
    • User-Defined Types:
      #include <sol/sol.hpp>
#include <cassert>

struct Vars {
    int boop = 0;
};

int main() {
    sol::state lua;
    lua.new_usertype<Vars>("vars", "boop", &Vars::boop);
    lua.script("beep = vars.new()\n" "beep.boop = 1");
    assert(lua.get<Vars>("beep").boop == 1);
}
