Stanford CoreNLP is an integrated framework that simplifies the application of various language analysis tools to text. With just two lines of code, you can run a suite of linguistic analyses on plain text. Here are some key features:

  1. Linguistic Annotations: CoreNLP allows you to derive linguistic annotations for text, including:

    • Token and sentence boundaries
    • Parts of speech
    • Named entities
    • Dependency and constituency parses
    • Coreference resolution
    • Sentiment analysis
    • Quote attributions
    • Relations

  2. Supported Languages: CoreNLP currently supports eight languages: Arabic, Chinese, English, French, German, Hungarian, Italian, and Spanish.

  3. Pipeline: The centerpiece of CoreNLP is the pipeline. Pipelines take raw text as input, apply a series of NLP annotators, and produce a final set of annotations. These annotations are accessible through a simple API and can be serialized to a Google Protocol Buffer.

  4. Usage:

    • Java: Stanford CoreNLP is written in Java, and recent releases require Java 8+. You can interact with it via the command-line, its original Java programmatic API, or third-party APIs for various programming languages.
    • Web Service: You can also use CoreNLP via a web service.
    • Operating Systems: It works on Linux, macOS, and Windows.

  5. License: The full Stanford CoreNLP is licensed under the GNU General Public License v3 or later.
