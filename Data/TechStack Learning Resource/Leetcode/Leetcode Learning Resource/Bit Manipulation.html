Bit manipulation in LeetCode involves using bitwise operators to perform operations on binary numbers. Some useful tricks for efficient coding include:

  1. To create a number with only the k-th bit set, use 1 << (k-1).
  2. To check whether the k-th bit is set or not, use n & (1 << (k - 1)).
  3. To set the k-th bit to 1, use n | (1 << (k - 1)).
  4. To clear the k-th bit, use n & ~ (1 << (k - 1)).
  5. To toggle the k-th bit, use n ^ (1 << (k – 1))1.
