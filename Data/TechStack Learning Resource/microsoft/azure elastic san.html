Azure Elastic SAN is a cloud-native service that provides a scalable, cost-effective, high-performance, and comprehensive storage solution for various Azure compute options. It offers the same simplified management experience and resiliency as on-premises storage area networks (SANs) with support for iSCSI protocols and zone redundant storage1.
