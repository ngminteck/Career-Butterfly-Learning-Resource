Microsoft Visio, formerly known as Microsoft Office Visio, is a diagramming and vector graphics application that allows users to create various types of diagrams, including flowcharts, org charts, floor plans, network diagrams, UML diagrams, and more. It is part of the Microsoft 365 family and has both web and desktop versions1.
