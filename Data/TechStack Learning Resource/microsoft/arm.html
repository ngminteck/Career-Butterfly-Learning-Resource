Certainly! In a nutshell, Azure Resource Manager (ARM) is the native platform for infrastructure as code (IaC) in Azure. It allows you to centralize the management, deployment, and security of Azure resources by grouping them into containers called resource groups1. Here are five free resources to learn more about ARM:

  1. Azure Resource Manager Overview: Dive into the basics and understand the consistent management layer provided by ARM.
  2. Azure Resource Manager Templates (ARM Templates): Learn how to create and deploy ARM templates to automate resource provisioning.
  3. Bicep Overview: Explore Bicep, a new language that simplifies ARM template creation.
  4. Azure DevOps Lab: Learning ARM Templates: A GitHub repository with tutorials and videos to guide you through best practices for ARM templates.
  5. Azure Custom Providers: Understand how to create custom providers and extend ARM capabilities.
