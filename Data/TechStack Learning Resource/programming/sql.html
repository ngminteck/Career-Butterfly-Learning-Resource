  3. Relational Database Management System (RDBMS):

    • RDBMS is the foundation for SQL and modern database systems.
    • Examples of RDBMS include MS SQL Server, IBM DB2, Oracle, MySQL, and Microsoft Access.
    • Data in RDBMS is organized into tables, which consist of columns (fields) and rows (records).
    • Each record represents an individual entry in the table, and each column maintains specific information about the records.
