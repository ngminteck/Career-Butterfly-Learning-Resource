Oracle Cloud is a comprehensive cloud computing service provided by Oracle Corporation. It offers a wide range of services, including:

  1. Infrastructure as a Service (IaaS): Oracle Cloud provides servers, storage, and network resources. This allows organizations to build and manage their infrastructure in the cloud. Notable offerings include:

    • Compute: Virtual Machine Instances, bare metal servers, and GPU servers.
    • Storage: Block volumes, file storage, object storage, and archive storage.
    • Networking: Configurable IP addresses, subnets, routing, and firewalls.
    • Governance: Features for auditing, identity management, and access control.
    • Database Management / Data Management: Solutions for database workloads, big data, and streaming workloads.

  2. Platform as a Service (PaaS): Oracle Cloud offers a development and deployment platform for creating and extending applications. It supports various open standards, open-source applications, and programming languages. Notable services include:

    • Kubernetes, Spark, Hadoop, Kafka, MySQL, and Terraform.
    • A variety of programming languages, databases, tools, and frameworks.

  3. Software as a Service (SaaS): Oracle Cloud hosts and manages business applications, including Human Capital Management, Enterprise Resource Planning, and Supply Chain Management.

  4. Data as a Service (DaaS): Oracle Cloud provides data-related services, such as analytics, machine learning, and deep learning.
