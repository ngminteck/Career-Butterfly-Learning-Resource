The Azure Quantum Development Kit (Modern QDK) is the software development kit (SDK) required to interface with the Azure Quantum service. Let me break it down for you:

  1. Purpose:

    • The Azure Quantum Development Kit (QDK) empowers developers to create quantum programs that can run on quantum hardware within the Azure Quantum ecosystem 1.
    • It provides the necessary tools and libraries for quantum programming.

  2. Components:

    • The Modern QDK includes:
      • Q#: A quantum programming language specifically designed for expressing quantum algorithms.
      • Accompanying Libraries: These libraries enhance the capabilities of Q# and facilitate quantum development.
      • Integrated Development Environments (IDEs): You can use different IDEs to write and execute quantum code.
        • Visual Studio Code (VS Code): Provides rich Q# language support, including features like IntelliSense and debugging. It also integrates with Azure for submitting jobs to quantum hardware.
        • Jupyter Notebooks: Available on the Azure Quantum portal, where you can create, upload, store, and run quantum code on simulators or actual quantum hardware targets.
        • Azure Quantum Website: Allows you to write and run Q# code directly in your browser using an online code editor. You can also open your code in VS Code on the web without any installation or Azure account requirements.
        • Local Setup: If you prefer, you can set up the Modern QDK locally on your machine.

  3. Supported Languages and Environments:

    • The QDK enables you to develop quantum applications using different languages and environments:
      • Q#: The primary language for quantum programming.
      • Python: Integrated support for Python.
      • Qiskit and Cirq: Libraries for quantum computing.
      • Hybrid Development: Combining classical and quantum components.
      • Resource Estimator: Helps estimate resource requirements for quantum programs.
      • QPU Access: Access to quantum processing units (requires an Azure subscription).
