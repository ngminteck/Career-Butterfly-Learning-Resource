SWIFT (Society for Worldwide Interbank Financial Telecommunications) is a crucial system that powers most international money and security transfers. Let‚Äôs delve into what SWIFT does and how it works:

  1. Purpose and Function:

    • SWIFT is a vast messaging network used by financial institutions to securely send and receive information related to money transfer instructions.
    • It enables individuals and businesses to make electronic or card payments even if the customer or vendor uses a different bank than the payee.
    • SWIFT is the largest and most streamlined method for international payments and settlements.

  2. How SWIFT Works:

    • Each member institution is assigned a unique ID code known as a Bank Identifier Code (BIC).
    • The BIC consists of either eight or eleven characters and provides information about the bank name, country, city, and branch.
    • For example, the Italian bank UniCredit Banca has the BIC code UNCRITMM:
      • First four characters: Institute code (UNCR for UniCredit Banca)
      • Next two characters: Country code (IT for Italy)
      • Next two characters: Location/city code (MM for Milan)
      • Last three characters (optional): Used to assign codes to individual branches.
    • When a customer wants to send money internationally, their local bank uses SWIFT to transmit a payment transfer message to the recipient‚Äôs bank.
    • The recipient bank receives the SWIFT message and processes the incoming payment, crediting it to the recipient‚Äôs account.

  3. Important Points:

    • SWIFT is not a financial institution; it facilitates secure communication between member institutions.
    • More than 11,000 global SWIFT member institutions send an average of 44.8 million messages daily through the network.
    • While SWIFT is crucial to global financial infrastructure, it does not hold or transfer assets directly.
